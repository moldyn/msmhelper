def openmicrostates(file_name, limits_file=None, **kwargs)

Load 1d file and split according to limit file.

Both, the limit file and the trajectory file needs to be a single column file. If limits_file is not provided it will return [traj]. The trajectory will of dtype np.int16, so the states needs to be smaller than 32767.

def opentxt(file_name, comment='#', nrows=None, **kwargs)

Open a text file.

This method can load an nxm array of floats from an ascii file. It uses either pandas read_csv for a single comment or as fallback the slower numpy laodtxt for multiple comments.

Warning: In contrast to pandas the order of usecols will be used. So if

using Â´data = opentxt(…, uscols=[1, 0])Â´ you acces the first column by data[:, 0] and the second one by data[:, 1].
